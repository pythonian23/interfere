# Interfere
A small raylib application to mess with interfering sine waves

## Keybindings
|                     Key | Function       |
|------------------------:|:---------------|
| Left/Middle/Right Click | Set point      |
|           Click + Space | Point/Axis/Off |
|                   Enter | Color view     |
